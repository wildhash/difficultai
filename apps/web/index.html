DifficultAI

High-Pressure Voice Training

Quick Start Guide:

  • Option 1: Use LiveKit Agents Playground at cloud.livekit.io/agents
  • Option 2: Deploy this web client with a backend that creates rooms and tokens
  • During session: Speak naturally, interrupt anytime (barge-in supported)
  • After session: Receive scorecard with 6 scores (1-10) + 3 coaching points
Note: This is a minimal reference client. For production use, you need to:
  1. Set up a backend to create LiveKit rooms
  2. Generate participant tokens with scenario metadata
  3. Update LIVEKIT_URL and implement getToken() function

Easiest way to test: Use LiveKit Cloud's Agents Playground with the scenario metadata JSON shown below.

0 = easy, 0.5 = moderate, 1 = maximum pressure
